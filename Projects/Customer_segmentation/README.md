# Customer Segmentation Analysis

## 📌 Problem Statement
Analyze customer purchasing behavior to identify different customer segments and improve business decisions.

## 🛠 Tools Used
- Power BI
- Excel

## 📊 Key Insights
- Top 20% customers contribute majority of revenue
- Repeat customers generate higher lifetime value
- Certain regions show higher purchase frequency
